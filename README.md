# BudgetApp
------------

#### DESCRIPTION
A Budget Counter App with:
  - Budget, Remaining and Spen so far.
  - Expenses list.
  - Add Expense with Name and Cost.

------------
#### USED TOOLS

- __***React***__
- __***Bootstrap***__
- __***UUIDv4 used to generate UUID's for generating ID's.***__
